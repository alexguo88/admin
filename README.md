# 环境

需要安装nodeJS环境

## 安装

npm install <---- 安装，如果下载比较慢，可使用淘宝的cnpm，请参考：https://npm.taobao.org/

## 开发服务器

npm run start <---- 使用npm命令运行

## 构建 

npm run build <---- 使用npm命令构建，构建完成后会在dist目录中
